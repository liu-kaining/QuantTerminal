## 📈 QuantTerminal：终极本地优先期权策略分析器

QuantTerminal 是一款**专业级、单文件 HTML 应用**，专为期权交易者和量化分析师设计。它允许您构建、模拟和分析复杂的**多腿期权策略**，**零设置**，并提供**完全隐私**。

---

### ✨ 主要功能

* **🚀 零安装：**
    * 一个单一的 `.html` 文件。**无需服务器、无需 Python、无需 Node.js**。只需下载并在任何现代浏览器中打开。
* **🔒 隐私至上：**
    * 所有计算都在您的浏览器**本地执行**。您的交易策略**绝不会离开您的设备**。
* **🧩 多腿策略构建器：**
    * 支持**看涨期权 (Call)**、**看跌期权 (Put)** 和**股票 (Stock)** 腿。
    * 内置预设：**长看涨**、**备兑看涨**、**牛/熊价差**、**跨式**、**蝶式**和**铁秃鹫**。
    * 完全可定制：添加**无限多**的腿来创建定制策略。
* **📊 高级风险分析：**
    * **希腊字母 (Greeks) 面板：** 监控**净 Delta、Gamma、Theta 和 Vega** 风险敞口。
    * **波动率输入：** 调整每条腿的 **IV (Implied Volatility - 隐含波动率)**，并输入**全局 HV (Historical Volatility - 历史波动率)** 上下文。
* **⚡ 交互式可视化：**
    * **动态盈亏 (P&L) 图表**，自动检测**盈亏平衡点 (BE)**。
    * **实时模拟滑块**，可视化特定到期价格下的盈亏。
* **🎨 专业用户界面 (UI)：**
    * “**黑暗金融科技 (Dark Fintech)**”美学，搭配定制构建的输入和下拉菜单，提供**无干扰的分析体验**。

---

### 🛠️ 如何使用

1.  下载 `index.html` 文件。
2.  **双击**在 Chrome、Edge 或 Safari 中打开。
3.  选择一个**策略预设**或**逐腿构建**您自己的策略。
4.  点击 "**Advanced (高级参数)**" 输入特定的希腊字母或波动率数据。
5.  **拖动价格滑块**以模拟潜在结果。
